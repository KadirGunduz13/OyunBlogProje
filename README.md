# 🎮 OyunBlogProje

Kullanıcıların oyunlarla ilgili yazılar paylaşabileceği, yorum yapabileceği ve içerik okuyabileceği dinamik bir blog platformudur. ASP.NET Web Application (.NET Framework) teknolojisi kullanılarak geliştirilmiştir.

## 🚀 Proje Hakkında

**OyunBlogProje**, oyun severler için bir yorum yapma platformudur. Kullanıcılar siteye giriş yaparak blog gönderilerine yorum yapabilir. Site aynı zamanda yönetici paneli ile içerik kontrolü sağlar.

## 🛠️ Kullanılan Teknolojiler

- ASP.NET Web Application (.NET Framework)
- C#
- ADO.NET (SQL bağlantısı)
- MS SQL Server
- HTML / CSS / JavaScript
- Entity Layer (Katmanlı Mimari)
- Master Page yapısı

## 📁 Proje Yapısı

- `OyunBlogProje.Entity` → Veri modelleri (Entity Layer)
- `OyunBlogProje` → UI ve sayfa mantıkları
- `Default.aspx` → Anasayfa
- `Login.aspx / Register.aspx` → Giriş ve kayıt işlemleri
- `AdminPanel.aspx` → Yönetici arayüzü
- `BlogDetay.aspx` → Gönderi detayları ve yorumlar

## 🔐 Özellikler

- ✅ Yazı paylaşımı (başlık, içerik, görsel)
- ✅ Yorum yapabilme
- ✅ Blog listeleme ve detay görüntüleme
- ✅ Yönetici onaylı gönderi sistemi
- ✅ SQL veritabanı ile dinamik veri yönetimi
